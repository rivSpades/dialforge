# Design spec

Brand name: DialForge. Domain, Twitter, and Instagram handles are all available.

Visual identity:
- Primary color: #1A1F36 (deep navy — dashboard background)
- Accent 1: #FF5C35 (warm orange — alerts, losses, CTAs)
- Accent 2: #00C896 (green — profit, success states)
- Display font: Space Grotesk (headings, numbers)
- Body font: Inter (readable at all sizes)

Tone: data-driven, scrappy, transparent, performance-obsessed, growth-minded.

Five key screens designed with full loading, empty, and error states:
1. Dashboard — Daily P&L Overview (spend vs revenue, margin %, sparkline toward $200 goal)
2. Campaign Manager (active campaigns with payout, traffic source, CPL, status toggle)
3. Live Call Feed (real-time ticker with duration, status, payout per call)
4. Journey Log — Weekly Timeline (chronological journal, color-coded green/red/yellow)
5. Goal Tracker — $200/Day Progress (progress ring, milestone markers at $50/$100/$150/$200)

All screens use skeleton loading states with pulsing navy cards. Error states show last-sync timestamps and cached data with warning badges. Empty states include clear CTAs to guide first actions.
